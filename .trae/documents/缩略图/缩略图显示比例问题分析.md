# 缩略图显示比例问题分析

## 问题1：缩略框中的图片不能完整展示，始终有裁切

### 问题分析
- ThumbnailGrid 尺寸：`Width="130" Height="110"`
- Grid.RowDefinitions：
  - Row 0：`Height="*"` - 图片区域
  - Row 1：`Height="20"` - 底部信息栏
- **实际图片区域高度**：110 - 20 = **90px**
- 解码尺寸：`ThumbnailWidth = 260px`

## 问题2：有的JPG图片预览图被上下挤压有黑边，比例与原图不对

### 问题分析

当前 ThumbnailService 中的关键问题：

1. **GetSystemThumbnailAsync 只设置了 DecodePixelWidth**
   ```csharp
   bitmap.DecodePixelType = DecodePixelType.Logical;
   bitmap.DecodePixelWidth = ThumbnailWidth; // 260px
   // 没有设置 DecodePixelHeight！
   ```

2. **系统缩略图可能已被裁切**
   - 使用 `GetThumbnailAsync(ThumbnailMode.PicturesView, 260)`
   - 系统可能返回的缩略图已经有黑边或被裁切

3. **ThumbnailHeight 配置未使用**
   - 定义了 `const int ThumbnailHeight = 80`
   - 但实际上没有使用！

## 修复方案

### 方案：完整移除解码尺寸限制
- 完全移除 `DecodePixelWidth` 和 `DecodePixelHeight` 的设置
- 让系统返回的缩略图保持原始比例
- 依靠 XAML 中 `Stretch="Uniform"` 来正确显示
