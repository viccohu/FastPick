# 底栏进度条展示区 - 实现计划

## [ ] 任务 1: 在 MainViewModel 中添加进度相关属性
- **Priority**: P0
- **Depends On**: None
- **Description**: 
  - 添加 `IsProgressVisible` 属性，控制进度条区域的显示/隐藏
  - 添加 `ProgressValue` 属性，存储当前进度值（0-100）
  - 添加 `ProgressText` 属性，存储进度说明文字（如"缩略图生成"）
  - 添加 `ProgressMaximum` 属性，存储进度最大值
  - 添加 `StartProgress(string description, int maximum = 100)` 方法开始进度
  - 添加 `UpdateProgress(int value)` 方法更新进度
  - 添加 `EndProgress()` 方法结束进度
- **Success Criteria**:
  - 属性能够正确通知 UI 更新
  - 进度值在 0-maximum 范围内
- **Test Requirements**:
  - `programmatic` TR-1.1: 验证属性变化通知正常工作
  - `human-judgment` TR-1.2: 验证进度值范围控制正确
- **Notes**: 使用 INotifyPropertyChanged 实现

## [ ] 任务 2: 在 MainPage.xaml 中添加进度条展示区 UI
- **Priority**: P0
- **Depends On**: 任务 1
- **Description**: 
  - 在底部工具栏上方添加一个新的 Grid 行
  - 创建进度条展示区容器，使用灰黑色主题配色
  - 添加 ProgressBar 控件，绑定到 `ProgressValue` 和 `ProgressMaximum`
  - 添加 TextBlock 显示进度说明文字，绑定到 `ProgressText`
  - 绑定 Visibility 到 `IsProgressVisible`
  - 设计配色方案：
    - 背景：`SurfaceBackgroundBrush` (#1E1E1E) 或 `ElevatedBackgroundBrush` (#2D2D2D)
    - 边框：`BorderBrush` (#333333)
    - 进度条颜色：`AccentBrush` (#27a0dc)
    - 文字颜色：`TextPrimaryBrush` (#FFFFFF)
- **Success Criteria**:
  - UI 样式符合灰黑色主题
  - 进度条正常显示和隐藏
  - 文字说明清晰可见
- **Test Requirements**:
  - `human-judgment` TR-2.1: 验证 UI 配色与主题一致
  - `human-judgment` TR-2.2: 验证显示/隐藏动画自然
- **Notes**: 使用现有主题资源，确保视觉一致性
